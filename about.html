Codash é uma Software House especializada no desenvolvimento de soluções inteligentes e criativas para sites WordPress.

Serviços

Plugins

  • Custom Post Types & Taxonomias personalizadas de acordo com suas necessidades
  • Dashboard com interface amigável para gerenciamento de "custom fields" e configurações gerais
  • Tabelas próprias no Banco de Dados
  • Versionamento constante via GIT
  • Testes de compatibilidade com o core do WordPress, e com outros plugins

Temas

  • Recebemos o seu layout em PSD, Adobe XD, Figma, entre outros formatos
  • Trabalhamos com Bootstrap, Bulma ou CSS estrutural personalizado
  • Desenvolvimento "Mobile First"
  • Temas otimizados p. SEO, com relatórios mensais do GTMetrix
  • Páginas de arquivo, autor, taxonomias, datas, home, buscas, erros etc.
  • Customizações via menu Aparências > Personalizar do WordPress

Integrações

  • WooCommerce, Elementor, BuddyPress, BBPress, Gutenberg, WPML, Traduções
  • Criação de rotas personalizadas na API REST do WordPress
  • Componentes REACT para Woocommerce Analytics
  • Blocos p. Elementos e Gutenberg
  • SMTP e disparos de e-mail
  • Importação de dados de ERP's que possuam API
